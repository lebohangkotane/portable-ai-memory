The Problem
Your AI memory is locked in a walled garden

Every conversation you have with an AI builds context — your preferences, skills, goals, style. But that context is trapped. Switch platforms and you start from zero.

🔒

Without PAM — Today's Reality

  • ChatGPT, Claude, Copilot — all siloed
  • Switch platform = lose all context
  • Rebuild your "relationship" from scratch
  • No standard way to move your memory
  • You don't own your conversation history
  • Vendors benefit from locking you in
🔓

With PAM — The Vision

  • One encrypted vault you own locally
  • Import from any AI platform
  • Carry your context to any new AI
  • Universal open schema — no lock-in
  • Per-platform privacy controls
  • Like email portability, but for AI

Phase 1 — What Was Built
The foundation is done

Phase 1 is the complete data layer — everything needed to ingest, store, search, and protect your AI memory. Validated against 407 real Copilot conversations.

📐
Universal Schema v1
Open JSON Schema defining memories, conversations, messages. 10 memory types. Platform-agnostic. Portable across any future AI.
🔐
Encrypted SQLite Vault
AES-256 encryption at rest. Zero cloud dependency. Full CRUD for conversations and memories. WAL mode for performance.
🔌
3 Import Adapters
ChatGPT (ZIP/JSON), Claude (JSON), Microsoft Copilot (CSV). Auto-detection. Plug-in interface to add new platforms in one file.
🧠
Memory Extraction Engine
Heuristic pattern matching pulls facts, preferences, skills, goals, and instructions from raw conversations. LLM extraction ready for Phase 2.
🔍
Semantic Search
Local sentence-transformer embeddings (no API key). Combined semantic + keyword scoring. Works fully offline.
🛡️
Privacy Controls
DENY ALL by default. Per-platform rules — Claude can see your skills, but not your sensitive memories. Memory-level access control.
⌨️
CLI Interface
pam import, pam search, pam list, pam stats, pam export. Rich terminal UI. Works on Windows, Mac, Linux.
🧪
32 Passing Tests
Vault CRUD, adapter detection & parsing, memory extraction accuracy, privacy filter enforcement. Zero warnings. CI-ready.
